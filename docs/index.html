Quick Start

SwiftTeX leverages KaTeX font metrics and web assembly for absolute precision.

npm install swifttex

Usage

import init, { render } from "swifttex";

// Initialize the WASM module
await init();

// Render to SVG
const { svg } = render("\\frac{x^2}{y}", { 
    display_mode: true, 
    output: "svg" 
});

document.getElementById("math").innerHTML = svg;

Plugin API

Register custom macros and symbols on the fly.

import { register_symbol } from "swifttex";

// Map \hbar to its unicode equivalent
register_symbol("hbar", "ℏ");
const result = render("\\hbar");
